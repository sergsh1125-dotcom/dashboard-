# 📊 Інтерактивний дашборд обліку забезпечення виробами

Аналітичний веб-дашборд для моніторингу рівня забезпечення виробами по регіонах України з інтерактивною картою, фільтрами та експортом звіту в Excel.

Проєкт реалізований на **Streamlit + Folium + Pandas**.

---

## 🚀 Можливості

### 🔎 Фільтрація

* по регіону
* по виробу

### 📈 KPI

* Загальна наявність
* Загальна потреба

### 📋 Таблиця по регіонах

* Потреба
* Наявність
* Нестача
* Надлишок
* % забезпечення

### 🗺 Аналітична карта України

* Постійні назви регіонів
* 5 рівнів забезпечення:

  * 🟩 ≥ 100%
  * 🟨 75–99%
  * 🟧 50–74%
  * 🟥 < 50%
  * ⬜ 0%
* Tooltip з % забезпечення
* Кастомна легенда

### 📥 Експорт

* Завантаження звіту в Excel одним кліком

## 📄 Формат CSV

Файл `example_stock.csv` повинен містити такі колонки:

| Назва колонки       | Опис                        |
| ------------------- | --------------------------- |
| region_name         | Назва регіону (українською) |
| product_name        | Назва виробу                |
| year_of_manufacture | Рік виробництва             |
| quantity            | Наявна кількість            |
| required_quantity   | Потреба                     |

> ⚠ Назви регіонів повинні співпадати з мапінгом у коді.

## 🧠 Логіка розрахунків

### % забезпечення:

```
(Наявність / Потреба) * 100
```

### Нестача:

```
max(Потреба - Наявність, 0)
```

### Надлишок:

```
max(Наявність - Потреба, 0)
```
